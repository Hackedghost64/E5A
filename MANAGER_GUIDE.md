# 🛠 Gupta Traders Bot: Manager's Task List

Please provide the following 2 items to the developer so the bot can be activated:

### 1. Google Gemini API Key
1. Go to **[Google AI Studio](https://aistudio.google.com/app/apikey)**.
2. Sign in with your Google Account.
3. Click **"Create API key"**.
4. Copy and send the key to the developer.

### 2. Instagram Page Access Token
1. Go to **[Meta for Developers](https://developers.facebook.com/)**.
2. Go to your App Dashboard -> **Instagram Graph API**.
3. Generate a **Page Access Token** for the linked Instagram account.
4. Copy and send the token to the developer.

### 3. Machine Catalog
If you need to change the list of machines the bot knows about, please update the list of names in the `data/catalog.json` file.
